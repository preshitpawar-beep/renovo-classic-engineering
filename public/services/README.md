# Service Images

Drop your photos here using these exact filenames so the site picks them up automatically:

- `engine.jpg` — for "Engine Work"
- `restoration.jpg` — for "Restorative Work"
- `maintenance.jpg` — for "Maintenance & Repair"
- `custom.jpg` — for "Custom Engineering"
- `laser.jpg` — for "Laser Cleaning"

**Recommended:**
- 4:3 aspect ratio (e.g. 1600×1200 or 1200×900)
- JPG format, under 500KB each
- Landscape orientation
- The card crops to fill, so leave a little breathing room around your subject

If a file is missing, the card will gracefully show a stylised number placeholder instead — so you can add them one at a time without breaking the layout.
